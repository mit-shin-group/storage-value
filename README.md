# The Value of Storage in Electricity Distribution: The Role of Markets

This repository supplements the article "The value of storage in electricity distribution: The role of markets" by Dirk Lauinger, Deepjyoti Deka, and Sungho Shin, available at [https://arxiv.org/abs/2510.12435](https://arxiv.org/abs/2510.12435). This project was funded by the [Future Energy Systems Center](https://energy.mit.edu/futureenergysystemscenter/) at the MIT Energy Initiative.
